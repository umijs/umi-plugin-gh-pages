# umi-plugin-gh-pages

umi plugin to publish to Github Pages.

## Usage

Configure in `.umirc.js`,

```js
export default {
  plugins: [
    ['umi-plugin-gh-pages', options],
  ],
}
```

## Options

Checkout https://github.com/tschaub/gh-pages#options
